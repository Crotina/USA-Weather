# Simple Weather App
# 天气预报程序

此天气app使用美国政府的官方API获取信息，对于美国以外的地区可能无法工作

To share a location, add `?latitude={latitude}&longitude={longitude}` at the end of url, and the website can detect the custom location and load this location

想要分享天气情况，添加`?latitude={latitude}&longitude={longitude}`至当前url末尾，但是你不需要手动拼接，程序可以自动拼接
